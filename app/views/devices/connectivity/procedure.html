Connectivity Procedure

Step-by-step procedure to connect your board to the dashboard (webserver):

Select one of the microcontroller chip from list below and connect it to the WIFI

Create a new device <%= link_to 'here', devices_path(category: :owned), style: 'color: red;' %> by clicking on "Add New Device"

Download the new device’s Credential files along with One supported set of libraries (provided on the dashboard in three languages i.e. Python, Node.js and Embedded C)

Save certificate files and libraries on the chip as suggested in particular documentation

After getting Internet access on the chip, connect it to the web server. (Given in the documentation.)

After a successful connection with the web server, you can control the chip from the Dashboard. When you “Switch On” the device (On dashboard), you will receive an “On” response on the chip's console. Similarly, when you “Switch Off” the device (On dashboard), you will receive an “Off"”response on the chip's console. You have to read the response and according to the value of the response, you have to control a LED bulb or Fan attached to the chip.
